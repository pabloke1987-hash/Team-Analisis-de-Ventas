## Escenario Elegido

### Escenario B – Análisis de Ventas de una Pequeña Empresa

Para el desarrollo del trabajo se seleccionó el escenario correspondiente al análisis de ventas comerciales, permitiendo implementar un pequeño sistema de procesamiento de datos mediante Python y la librería pandas.

---

## Descripción del Proyecto

El proyecto permite analizar un conjunto de datos simulados de ventas comerciales con el fin de obtener indicadores básicos relacionados con el desempeño de ventas de una empresa.

Entre las funcionalidades implementadas se encuentran:

- lectura de archivos CSV
- cálculo de ventas totales
- identificación del producto más vendido
- generación de resultados exportables
- organización estructurada del proyecto mediante Git y GitHub
- trazabilidad de tareas utilizando Jira

Además, se aplicaron prácticas de trabajo colaborativo mediante ramas de desarrollo, commits trazables y Pull Requests.

---

## Dataset Utilizado

Se utilizó un dataset simulado de ventas comerciales almacenado en el archivo:

/datos/ventas.csv

El dataset contiene información relacionada con:

- fecha de venta
- producto
- cantidad vendida
- precio unitario

Los datos fueron utilizados para realizar cálculos y análisis estadísticos básicos utilizando Python y pandas.
